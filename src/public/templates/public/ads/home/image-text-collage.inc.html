{% load i18n %} {% comment %} Generates a collage with two columns of images using srcsets with 1x and 2x images, next to text. An image can be portrait or landscape by giving it a modifier class of `dmb-img-text-collage__img--portait` or `dmb-img-text-collage__img--landscape` respectively. Valid combinations: 1 portrait in each column 1 landscape in each column 2 lanscape in each column 2 lanscape in one column and 1 potrait in the other. Each images should be given a unique class and then it's background-image should be set using 1x and 2x images in the tenant SCSS file as follows: .dmb-img-text-collage__img--1 { background-image: url('../img/retail/home-collage-img1.jpg'); @media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) { background-image: url('../img/retail/home-collage-img1@2x.jpg'); } } The default background properties are `background-position: center center` and `background-size: cover`, but can be overridden in the tenant SCSS file if required. {% endcomment %}
