{% comment %} Heading with a preceding SVG. The SVG can be overlayed by a number that increments whenever a new instance is placed on a page. To do this add `numbered=True` to the include and set `counter-reset: step-heading;` in the page SCSS. The number's default size and color are 13px and white. These can be overridden in the tenant SCSS file using `.dmb-step-heading__icon::after`. Example: {% include 'public/news/inc/step-heading.inc.html' with text='Find out more' svg='shield' numbered=True %} {% endcomment %}
